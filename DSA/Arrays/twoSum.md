Data Structure -> Arrays and Hashmap
Pattern -> Hashmap lookup
Brute force -> Check every possible pair using nested loop
Optimised ->
Store visited number in hashMap.
For each number, check if the difference exists in map, then return the index and index stored in map for difference.

Time complexity o(n) (Loop)
Space complexity o(n) (Hashmap can store n number of keys)

Edge cases: None

Clues: Unsorted array, lookups
